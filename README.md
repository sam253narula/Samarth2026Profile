# Samarth Narula — Cuberto-Inspired Profile Website

A premium static profile website for Samarth Narula, inspired by the animation style and editorial feel of Cuberto while using original content, layout, copy and visual assets.

## Stack

- Next.js App Router
- Tailwind CSS
- Framer Motion
- Lucide React
- Static export support

## Sections

- Animated hero with custom cursor and magnetic CTA buttons
- Moving marquee
- Impact metrics
- Product-style selected work cards
- Services / consulting capabilities
- Full professional journey from Cyberender LLC to Accenture
- Skills, certifications, publications and recognition
- Contact form, email, phone and WhatsApp CTA

## Local setup

```bash
npm install
npm run dev
```

Open:

```bash
http://localhost:3000
```

## Static build

```bash
npm run build
```

The static export will be generated inside the `out` folder.

## Resume

The resume PDF is included in `public/Samarth-Narula-Resume.pdf` and linked from the hero/footer.
